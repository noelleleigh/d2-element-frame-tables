# d2-element-frame-tables

Generate tables of all the elemental damage type and weapon frame combinations available in Destiny 2.

1. Set `API_KEY` and `USER_AGENT` in a `.env` file.
1. `npm install`
1. `npm start`
1. Open `index.html` in browser
